Visual Studio es un programa de coding que no necesita internet para usarlo, con esta applicacion puedes hacer cosas increibles, como una pagina web, videojuegos, etc.

La consola es un elemento que todas las paginas web, videojuegos, etc. Este es un elemento esencial para programacion. Se usa para escribir cosas en la consola y para los codigos ASCII, etc. Para abrirla solo tienes q hacer click derecho e inspeccionar y ya lo abres!
